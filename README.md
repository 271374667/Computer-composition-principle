# Computer-composition-principle
==中文名：计算机组成原理==

个人对于每章大小的说明：

| 章节\说明 | 字数 | 作用 |
| :---- | :----: | ---- |
| 小章 | 小于50行 | 一般是解释说明或者目录类 |
| 中章 | 大于50行但是小于100行 | 一般是课程的补充或者扩展 |
| 大章 | 大于100行但是小于300行 | 一般是正课的笔记 |
| 超大章 | 大于300行 | 一般多为成体系不可分割，或者重点难点 |
